# woogi-blog-assets

Cover image assets for [woogi.me](https://woogi.me) — a blue team / SOC analyst blog.

## Structure

```
thumbnails/   # Open Graph cover images (1200x630 PNG) for blog posts
```

## Usage

Cover image URLs follow this pattern:

```
https://raw.githubusercontent.com/Jung0310-bit/woogi-blog-assets/main/thumbnails/{slug}.png
```

Example:

```
https://raw.githubusercontent.com/Jung0310-bit/woogi-blog-assets/main/thumbnails/phishing-analysis-blue-team-guide.png
```
